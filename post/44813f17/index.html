补充

  1. 如需定义分式，可以使用\frac{}{}，如下：

    x2y2/31x(1)\frac{x^2}{y^{2/3}} \qquad \frac{1}{x} \tag{1} y2/3x2​x1​(1)

  2. 如果需要给公式编号，可以使用\tag{}，如上例

  3. 如果需要定义高次根式，可以使用\sqrt[]{}，其中[]是幂指数，{}是求根的真值：

2x2+y2+z24\sqrt{2} \qquad \sqrt[4]{x^2 + y^2 + z^2} 2​4x2+y2+z2​

  1. 可以使用\overline，\underline在表达式上方和下方添加横线，也可以使用\overbrace或\underbrace在表达式的上方或下方添加花括号：

    a‾A∪B‾a1+a2+⋯+an⏞nb1×b2×⋯×bn⏟n\underline{a} \qquad \overline{A \cup B} \\ \overbrace{a_1 + a_2 + \dots + a_n}^{n} \qquad \underbrace{b_1 \times b_2 \times \dots \times b_n}_{n} a​A∪Ba1​+a2​+⋯+an​​n​nb1​×b2​×⋯×bn​​​

  2. 可以使用\qquad生成空格隔开公式的某些部分，也可以用\text{}添加正常的文字描述，使用\\表示换行：

    abI’m some commentsa \qquad b\\ \text{I'm some comments} abI’m some comments

  3. 可以使用\begin{cases}和\end{cases}定义类似分段函数的表达式：

    f(x)={x2+1if x≥0ln⁡∣x∣otherwisef(x) = \begin{cases} x^2 + 1 &\text{if } x \ge 0 \\ \ln |x| &\text{otherwise} \end{cases} f(x)={x2+1ln∣x∣​if x≥0otherwise​

    花括号中的对象可以替换：

    • (p/b/v/V/B)matrix，表示矩阵；

      E=(100010001)E = \begin{pmatrix} 1 & 0 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & 1 \end{pmatrix} E=​100​010​001​​

    • euqations， 可以（带换行）输入连等式（隐式携带\tag，可以显式覆盖）；

      a=b+c=e+f\begin{equation} \begin{split} a &=b+c\\ &=e+f \end{split} \end{equation} a​=b+c=e+f​​​

    • 还有很多用法，详见KaTeX - Enviroments。

  4. 一般LaTeX渲染的字母是偏向手写体的。但是在一元微分中为了表示区别，微分符号应该使用\mathrm修饰

    f′′(x)=d2ydx2f''(x) = \frac{\mathrm d^2 y}{\mathrm d x^2} f′′(x)=dx2d2y​

  5. 常用的、有专用符号表示的基本初等函数，在LaTeX中有形式统一的表示方法，例如：

    sin⁡x+2cosh⁡x−3arctan⁡x+4ln⁡x−axlg⁡x\sin x + 2 \cosh x - 3 \arctan x + 4 \ln x - a^x \lg x sinx+2coshx−3arctanx+4lnx−axlgx
