Preface

空间几何主要是空间中曲线、曲面、空间的位置关系的表达与计算，为下一篇积分的计算做铺垫。

空间中的曲面

平面

平面是曲面的一个特例。

空间中任何一个平面都可以使用三元一次方程表示：

π:Ax+By+Cz+D=0(1-1)\pi: A x + B y + C z + D = 0 \tag{1-1} π:Ax+By+Cz+D=0(1-1)

其中，称向量n⃗=(A,B,C)\vec n = (A, B, C)n=(A,B,C)是平面的法向量，总有n⃗⊥π\vec n \perp \pin⊥π。类似于平面直线的点斜式，可以写出空间方程的点法式方程：

π:A(x−x0)+B(y−y0)+C(z−z0)=0(1-2)\pi: A(x - x_0) + B(y - y_0) + C(z - z_0) = 0 \tag{1-2} π:A(x−x0​)+B(y−y0​)+C(z−z0​)=0(1-2)

若令(1−1)(1-1)(1−1)中x,y,zx, y, zx,y,z任意两个未知数为零，则可以得到空间平面的截距式方程：

π:xa+yb+zc=1(1-3)\pi: \frac x a + \frac y b + \frac z c = 1 \tag{1-3} π:ax​+by​+cz​=1(1-3)

根据“不共线的三点共面”的原理，平面也可以表示为三点式：

π:∣x−x1y−y1z−z1x−x2y−y2z−z2x−x3y−y3z−z3∣=1(1-4)\pi: \begin{vmatrix} x - x_1 & y - y_1 & z - z_1 \\ x - x_2 & y - y_2 & z - z_2 \\ x - x_3 & y - y_3 & z - z_3 \\ \end{vmatrix} = 1 \tag{1-4} π:​x−x1​x−x2​x−x3​​y−y1​y−y2​y−y3​​z−z1​z−z2​z−z3​​​=1(1-4)

常见的曲面及其表达式

空间曲面的表达式一般为F(x,y,z)=0F(x, y, z) = 0F(x,y,z)=0。一般着重研究二次曲面，即三个坐标的最高幂次为二次的曲面。

常见的二次曲面主要有：

  1. 椭球面

    x2a2+y2b2+z2c2=1(1-5)\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1 \tag{1-5} a2x2​+b2y2​+c2z2​=1(1-5)

    该方程满足：

    • abc≠0abc \neq 0abc=0；
    • a=b=c≠0a = b = c \neq 0a=b=c=0时，该曲面时一个球面；
    • 该曲面在三个坐标轴上的截距分别为±a,±b±c\pm a, \pm b \pm c±a,±b±c。

  2. 单叶双曲面

    x2a2+y2b2−z2c2=1(1-6)\frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = 1 \tag{1-6} a2x2​+b2y2​−c2z2​=1(1-6)

    可以视为双曲线x2a2−z2c2=1\frac{x^2}{a^2} - \frac{z^2}{c^2} = 1a2x2​−c2z2​=1绕zzz轴（虚轴）旋转一周形成的曲面。

  3. 双叶双曲面

    x2a2−y2b2−z2c2=1(1-7)\frac{x^2}{a^2} - \frac{y^2}{b^2} - \frac{z^2}{c^2} = 1 \tag{1-7} a2x2​−b2y2​−c2z2​=1(1-7)

    可以视为双曲线x2a2−y2b2=1\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1a2x2​−b2y2​=1绕yyy轴（实轴）旋转一周形成的曲面。

  4. 椭圆锥面

    x2a2+y2b2−z2c2=0(1-8)\frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = 0 \tag{1-8} a2x2​+b2y2​−c2z2​=0(1-8)

  5. 椭圆抛物面

    x22p+y22q=z,p>0,q>0(1-9)\frac{x^2}{2p} + \frac{y^2}{2q} = z, p \gt 0, q \gt 0 \tag{1-9} 2px2​+2qy2​=z,p>0,q>0(1-9)

如果令α=(xyz)\alpha = \begin{pmatrix} x \\ y \\ z \end{pmatrix}α=​xyz​​，则二次型可以表示为f(x,y,z)=αTAαf(x, y, z) = \alpha^T A \alphaf(x,y,z)=αTAα。结合上面的表达式，可以看出曲线(1−5)(1-5)(1−5)至(1−8)(1-8)(1−8)都满足二次型的特征[1], 其中只有封闭曲面的方程是正定二次型。

除了二次曲面之外，当曲面方程缺失一项时，形成柱面。常见的柱面主要有椭圆柱面、双曲柱面和抛物柱面。

空间中的曲线

曲线是由两个曲面相交形成的。

Γ:{F(x,y,z)=0G(x,y,z)=0(2-1)\Gamma : \begin{cases} F(x, y, z) = 0 \\ G(x, y, z) = 0 \end{cases} \tag{2-1} Γ:{F(x,y,z)=0G(x,y,z)=0​(2-1)

曲线也可以使用参数方程表示：

Γ:{x=φ(t)y=ψ(t)z=ω(t),α<t<β(2-2)\Gamma: \begin{cases} x = \varphi(t) \\ y = \psi(t) \\ z = \omega(t) \end{cases}, \alpha \lt t \lt \beta \tag{2-2} Γ:⎩⎨⎧​x=φ(t)y=ψ(t)z=ω(t)​,α<t<β(2-2)

直线

直线是曲线的一个特例。

在任何时候，直线都可以看作是两个平面相交得来的，因此直线的一般式方程可以表示为：

l:{A1x+B1y+C1z+D1=0A2x+B2y+C2z+D2=0(2-3)l: \begin{cases} A_1 x + B_1 y + C_1 z + D_1 = 0 \\ A_2 x + B_2 y + C_2 z + D_2 = 0 \end{cases} \tag{2-3} l:{A1​x+B1​y+C1​z+D1​=0A2​x+B2​y+C2​z+D2​=0​(2-3)

在直角坐标系中，直线的方程由其方向向量和直线上一点的坐标共同确定，这种方程称为点向式。不妨设直线的方向向量为τ⃗=(l,m,n)\vec \tau = (l, m, n)τ=(l,m,n)，直线上一点的坐标为P(x0,y0,z0)P (x_0, y_0, z_0)P(x0​,y0​,z0​)，那么点向式可以表示为：

l:x−x0l=y−y0m=z−z0n(2-4)l: \frac{x - x_0}{l} = \frac{y - y_0}{m} = \frac{z - z_0}{n} \tag{2-4} l:lx−x0​​=my−y0​​=nz−z0​​(2-4)

直线的参数方程是一组一次方程：

l:{x=lt+x0y=mt+y0z=nt+z0,α<t<β(2-5)l: \begin{cases} x = l t + x_0 \\ y = m t + y_0 \\ z = n t + z_0 \end{cases}, \alpha \lt t \lt \beta \tag{2-5} l:⎩⎨⎧​x=lt+x0​y=mt+y0​z=nt+z0​​,α<t<β(2-5)

又因为两点确定一直线，那么直线的两点式可以写作：

l:x−x1x2−x1=y−y1y2−y1=z−z1z2−z1(2-6)l : \frac{x - x_1}{x_2 - x_1} = \frac{y - y_1}{y_2 - y_1} = \frac{z - z_1}{z_2 - z_1} \tag{2-6} l:x2​−x1​x−x1​​=y2​−y1​y−y1​​=z2​−z1​z−z1​​(2-6)

经过仔细观察可以看出，参数方程和两点式都是直接由点向式推导得出。下面简单介绍一下一般式到点向式的转换过程。

一般式转换为点向式

从空间关系上，构成直线一般式的两个平面方程的法向量和直线的方向向量垂直，即直线的方向向量是两个平面的法向量的外积：

τ⃗=(A1,B1,C1)×(A2,B2,C2)(2-7)\vec \tau = (A_1, B_1, C_1) \times (A_2, B_2, C_2) \tag{2-7} τ=(A1​,B1​,C1​)×(A2​,B2​,C2​)(2-7)

将式(2−3)(2-3)(2−3)作为非齐次线性方程组Aα=bA \alpha = bAα=b求解，显然此方程组满足r(A)=r(A∣b)=2<3r(A) = r(A | b) = 2 < 3r(A)=r(A∣b)=2<3，有无穷多解。解出直线上一点，和τ⃗\vec \tauτ代入式(2−3)(2-3)(2−3)即可把一般式转换为点向式。

曲线的投影

以投影到xOyxOyxOy平面为例，通过对(2−1)(2-1)(2−1)式进行变换，得到φ(x,y)=0\varphi(x, y) = 0φ(x,y)=0，则曲线Γ\GammaΓ在xOyxOyxOy平面上的投影就包含在曲线{φ(x,y)=0z=0\begin{cases}\varphi(x, y) = 0 \\ z = 0 \end{cases}{φ(x,y)=0z=0​中。

空间中的位置关系及其运算

设曲面（曲线）上某点的坐标为P0(x0,y0,z0)P_0 (x_0, y_0, z_0)P0​(x0​,y0​,z0​)。
